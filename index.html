Screeny is a tool for screen size and resolution comparison. Just select the resolutions you want to compare in the RESOLUTIONS menu.

Created by Yuri Goytacaz | @yurigoytacaz

Requirements

Screeny has only been tested and confirmed to work in Chrome and Firefox 4.
